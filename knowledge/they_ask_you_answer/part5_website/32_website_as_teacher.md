# They Ask, You Answer – Kapitola 32: Web ako učiteľ

## Kontext kapitoly
Kapitola tvrdí, že web má byť najlepší „učiteľ“ na trhu. Nie brožúra, nie billboard, ale platforma, ktorá vedie kupujúceho krok za krokom k rozhodnutiu.

## Kľúčové myšlienky
- web má odpovedať, učiť a vysvetľovať
- zákazník chce objektívne informácie, nie marketingové frázy
- web má imitovať kvalitný predajný rozhovor
- výsledkom je kratší cyklus a menej odporov

## Modely, princípy, procesy
- jadrom webu je Learning Center (content hub)
- každá stránka má odpovedať na 1 jasnú otázku
- web musí byť orientovaný na problémy, nie na seba
- Big 5 sekcie ako súčasť hlavnej navigácie

## Využitie pre TRIAS Edge
- kľúčový princíp pre budovanie klientských webov
- agent môže auditovať web podľa „učiteľskej“ kvality obsahu
- ideálne pre firmy v technických odvetviach

## Story a príklady
Web River Pools obsahuje desiatky článkov, ktoré suplujú prvé tri stretnutia so zákazníkom.

## Jednovetová esencia
Web má učiť spôsobom, ktorý predbehne predajcu o tri kroky dopredu.
